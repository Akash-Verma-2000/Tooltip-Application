# Tooltip Application

Presenting a sleek tooltip application built with HTML, CSS, JavaScript, and React.js. Users can effortlessly control tooltip direction—top, bottom, left, or right—enhancing UX without compromising content visibility. This lightweight tool seamlessly integrates into any project, offering a streamlined user experience and unparalleled flexibility. Elevate your design with precision-placed tooltips, optimizing engagement and usability.

## Steps to Setup

1. Clone the repository.
2. Install dependencies **`npm i`**
3. Run the server **`npm start`**
